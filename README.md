# Sales Data Analysis Using Python

## Overview
This project demonstrates how Python can be used to clean, analyze, and visualize sales data to uncover business insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Key Analysis
- Data cleaning and preprocessing
- Sales aggregation by region and product
- Trend analysis over time
- Visual exploratory analysis

## Files
- sales_analysis.ipynb – Jupyter Notebook containing the analysis
- data/sales_data.xlsx – Dataset used for analysis
